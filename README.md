#Kronos Pro Remote

Kronos Pro Remote is a program add-on for **Kodi 21.x (Omega)** designed for **LibreELEC** and **Windows** systems. 
It provides a professional-grade remote control experience for Kodi, with a focus on precision, responsiveness, and system stability.

---

## Features
- **Low-Latency Input** – Fast and accurate command execution for smooth navigation. 
- **Advanced Key Mapping** – Supports custom remote layouts and button behavior. 
- **Navigation Engine** – Optimized directional control for media lists, menus, and overlays. 
- **Input Stability Layer** – Prevents duplicate or stuck commands during heavy UI load. 
- **Background Processing** – Lightweight event listener with minimal system impact. 
- **Refined User Experience** – Designed to feel instant, clean, and reliable under all load conditions.

---

## Compatibility
- Kodi **21 Omega** 
- LibreELEC 12.0.2+ 
- Windows 10/11 
- Arctic Zephyr Reloaded skin 
- Python 3 (bundled with Kodi)
